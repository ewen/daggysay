# Daggy Say

## Usage

`daggysay "Profound statement"`

To get a quote of the day with every login, install [fortune](https://en.wikipedia.org/wiki/Fortune_(Unix)) and add to your rc file:

`daggysay "$(fortune)"`

## Example

> daggysay "$(fortune)"
